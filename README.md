# Методы Super Resolution и Frame Interpolation в применении к видео

Создание классической системы видео-аналитики, включающей в себя детектор объектов, визуальный трекер (VOT), реидентификатор (REID), и собственно, трекер объектов - главный алгоритм системы, который объединяет компоненты в единое целое. Трекер занимается сопоставлением наблюдений, завязкой траекторий, ведением галлереи REID сигнатур и реализует логику восстановления потерянных объектов. Также на трекере задача приоритезации запросов к REID и VOT модулям. Спроектируем систему контроля качества и обсудим метрики понятные для заказчика. Познакомимся с системой разметки CVAT. Рассмотрим джентельменский набор функциональности, который обычно нужен заказчику: пересечение линий и счетчик объектов, вторжение в зону интереса, определение времени пребывания в зоне, тепловая карта, прочее. Основной фокус проекта - не нейросетевые модели, а то, что с ними делать. Впрочем, особенности их обучения тоже будут затронуты. Задача - научиться делать конечный продукт в мире CV.
