# Методы Super Resolution и Frame Interpolation в применении к видео

[Канбан-доска](https://github.com/users/mshsvnv/projects/1)

## Описание

Проектирование и тренировка модуля, который решает задачу повышения частоты кадров или разрешения видео. Познакомимся с оптическим потоком и теоремой Котельникова, некорректно поставленными задачами и как это может повлиять на выбор архитектуры, внутреннего представления данных и штрафных функций. Предлагается реализовать систему сбора и подготовки данных (закачка, фильтрация и балансировка), проработать архитектуру моделей, создать систему тренировки модели и оценки качества.

## Запуск

В папке [code](./code/)

1. Активация виртуальной среды
```bash
source ./venv/Scripts/activate
```

2. Установка зависимостей
```bash
pip install -r requirements.txt
```

3. Запуск ```main.py```
```bash
python main.py 'input.mp4' 'output.mp4'
```
